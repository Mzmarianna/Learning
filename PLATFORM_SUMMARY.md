# Platform Summary

Learning Kingdom blends game loops with neurodivergent-informed tutoring.

## Pillars

- **Gamified mastery:** Tracks, streaks, and achievements that respect sensory pacing.
- **Community co-play:** Multiplayer squads, collaborative quests, and leaderboards.
- **Family analytics:** Guardian dashboards surface the right coaching moments.
- **Hybrid delivery:** Lightweight PWA shell with Supabase functions for realtime services.

## Technical stack

- Vite + React front-end with TypeScript.
- Netlify hosting for static assets and edge deploys.
- Supabase for auth, database, and custom Deno edge functions.
- Scripts folder provides repeatable deployment and testing commands.
