# Features Comparison

| Capability | Learning Kingdom | Typical LMS |
| --- | --- | --- |
| Neurodivergent-informed tutoring | Strength-based quests, sensory breaks, real-time streak coaching | One-size-fits-all lesson pacing |
| Gamification depth | Achievements, co-op quests, streak forecasts | Badges or points without adaptive feedback |
| Family transparency | Guardian dashboard with growth, regulation, and mentor notes | Progress percentages without actionable context |
| Multiplayer support | Peer squads, leaderboards tuned for collaboration | Individual progress tracking only |
| Deployment footprint | PWA-ready front-end + Supabase Edge | Monolithic web app |
