## Color Scheme

**Primary Colors:**
- Purple: `from-purple-600` to `via-pink-600` to `to-cyan-600`
- Accent: Pink, Cyan, Yellow
- Background: Gradient from purple-50 via white to cyan-50

**Component Styles:**
- Cards: White background with 2px purple border, rounded corners
- Buttons: Gradient backgrounds with shadow effects
- Icons: Lucide React icons in matching colors
- Text: Gray-900 for headings, Gray-700 for body

## Responsive Design

All pages are fully responsive with:
- Mobile-first approach
- Breakpoints at `sm:`, `md:`, `lg:`
- Stacked layouts on mobile
- Grid layouts on desktop
- Touch-friendly button sizes

## Accessibility

- Semantic HTML elements
- Proper heading hierarchy
- Color contrast ratios meet WCAG AA
- Focus states on interactive elements
- Screen reader friendly text

## Animation & Motion

- Fade-in animations on page load
- Scale animations for icons
- Hover effects on buttons
- Smooth transitions between states
- Motion respects user preferences
