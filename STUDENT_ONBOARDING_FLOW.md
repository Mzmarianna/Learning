# Complete Student Onboarding Flow - End-to-End User Journeys

## Overview

This document describes the complete end-to-end experience for students, parents, tutors, and admins in the Mz. Marianna's Academy platform, from initial intake form submission through ongoing progress tracking.

---

## 1. STUDENT JOURNEY

### Phase 1: Onboarding (Parent-Initiated)
**Student is passive in this phase**

1. **Form Submission**
   - Parent fills out Google Form questionnaire
   - Student's information is captured (name, grade, struggles, interests, strengths)
   - Form automatically saves to Google Sheets

2. **Profile Creation**
   - System automatically creates student profile
   - Assigned to appropriate tier based on grade:
     - **Early Explorers** (K-2nd grade)
     - **Explorers** (3rd-5th grade)  
     - **Warriors** (6th grade+)
   - Initial preferences set based on interests/strengths
   - Login credentials prepared

### Phase 2: Assessment
**Student becomes active participant**

3. **Welcome Email**
   - Student receives welcome email with:
     - Login credentials
     - Assessment scheduling link
     - Introduction to Wowl (AI mascot)
     - Preview of the Learning Kingdom

4. **Initial Assessment (25-30 minutes via Zoom)**
   - Student meets tutor
   - Takes fun, game-based placement test
   - Conversation feels natural, not like a test
   - Questions adapt based on student's answers
   - Common Core standards assessed for grade level

5. **Results & Placement**
   - Student placed at appropriate level within their tier
   - Starting quests assigned
   - Initial XP and gems awarded (welcome bonus)
   - Two starter characters unlocked (Wise Owl, Sparkle Unicorn)

### Phase 3: Active Learning
**Daily/weekly engagement**

6. **Dashboard Login**
   - Lands on personalized Learning Kingdom dashboard
   - Sees current quest with story narrative
   - Views available challenges (activities)
   - Checks progress bar and XP
   - Sees unlocked badges and characters

7. **Quest Engagement**
   - Selects a challenge from current quest
   - Reads Wowl's encouraging introduction
   - Completes learning activity (may involve):
     - Math problems in Roblox
     - Reading comprehension games
     - Creative projects
     - Video submissions
     - Drawing/writing tasks

8. **Submission & Feedback**
   - Submits work through platform
   - Wowl AI provides instant encouragement
   - Waits for tutor assessment (usually 24-48 hours)
   - Receives detailed feedback with:
     - Mastery level (Emerging → Mastered)
     - XP earned
     - Specific praise for strengths
     - Guidance for improvements
     - Badge unlocked (if challenge completed)

9. **Progress & Rewards**
   - XP accumulates → Level up
   - Gems earned → Unlock new characters
   - Badges collected → Portfolio grows
   - Quests completed → New quests unlock
   - Achievements celebrated with animations

### Phase 4: Continuous Learning
**Ongoing cycle**

10. **Weekly Rhythm**
    - Monday: New quest available
    - Throughout week: Complete challenges
    - Friday: Weekly progress report
    - Weekend: Optional bonus challenges

11. **Parent Touchpoints**
    - Weekly progress email to parent
    - Parent can view dashboard
    - Celebrate achievements together
    - Discuss areas for growth

12. **Tutor Interaction**
    - 1-on-1 tutoring sessions (if enrolled)
    - Feedback on submissions
    - Adjustment of difficulty
    - Celebration of milestones

## 2. PARENT JOURNEY

### Phase 1: Discovery & Intake

1. **Initial Contact**
   - Discovers Mz. Marianna's Academy (website, referral, ad)
   - Clicks "Get Started" on homepage
   - Lands on Welcome Page

2. **Questionnaire**
   - Fills out comprehensive intake form:
     - Child's name and grade
     - Learning struggles
     - Strengths and interests
     - Program preferences
     - Availability schedule
     - Payment preferences
     - Questions for Marianna
   - Form saves to Google Sheets automatically

3. **Confirmation**
   - Receives immediate confirmation email:
     - Thank you message
     - Next steps outlined
     - Link to schedule assessment
     - Timeline expectations

### Phase 2: Assessment & Onboarding

4. **Assessment Scheduling**
   - Clicks Google Calendar link
   - Books 25-30 minute assessment slot
   - Receives calendar invitation with Zoom link
   - Gets reminder email 24 hours before

5. **Pre-Assessment Communication**
   - Email with what to expect:
     - "Make it fun, like a game"
     - No pressure environment
     - Privacy of Zoom space
     - Materials needed (optional)

6. **Assessment Day**
   - Student takes assessment
   - Parent can observe (optional)
   - Usually 25-30 minutes
   - Feels conversational, not test-like

7. **Results & Action Plan ($30 includes)**
   - Receives detailed action plan within 48 hours:
     - Specific strengths identified
     - Areas for improvement
     - Recommended tier/level placement
     - Suggested focus areas
     - Common Core standards assessed
   - Clear next steps presented

### Phase 3: Decision & Enrollment

8. **Program Choice**
   - **Option A: Teach at Home**
     - Download action plan
     - Receive resource recommendations
     - Access to some free materials
     - Can sign up for tutoring later
   
   - **Option B: Enroll in Tutoring**
     - Choose program:
       - Roblox Math ($30/session or $480/16 weeks)
       - Reading Tutoring ($30/session or $480/16 weeks)
       - Homeschool Program ($120/week)
       - Summer Camp ($250/week or $2500/10 weeks)
     - Select payment method (ClassWallet, Credit Card)
     - Complete enrollment

9. **Setup & Access**
   - Student receives login credentials
   - Parent gets access to parent dashboard
   - Both receive platform orientation email
   - First tutoring session scheduled (if applicable)

### Phase 4: Ongoing Monitoring

10. **Weekly Updates**
    - Automated email every Friday:
      - Quests completed this week
      - XP earned and level progress
      - Badges unlocked
      - Time spent learning
      - Areas of strength shown
      - Recommended support areas

11. **Parent Dashboard**
    - Real-time access to:
      - Student's current quest progress
      - All submissions and feedback
      - Mastery levels by subject
      - Engagement metrics
      - Upcoming sessions
      - Communication with tutor

12. **Communication**
    - Message tutor directly through platform
    - Receive notifications for:
      - New submissions assessed
      - Level ups
      - Milestones reached
      - Concerns flagged
      - Session reminders

13. **Progress Reports**
    - Monthly detailed reports:
      - Skills mastered
      - Growth areas
      - Time invested
      - Engagement levels
      - Recommended next steps
      - Celebration of achievements

### Phase 5: Continued Partnership

14. **Adjustment Opportunities**
    - Can change tutoring frequency
    - Upgrade/downgrade programs
    - Request different tutor if needed
    - Add siblings
    - Pause service

15. **Milestone Celebrations**
    - Completion of quest series
    - Tier advancement
    - Special achievements
    - End of program celebrations

## 3. TUTOR/ADMIN JOURNEY

### Phase 1: New Student Intake

1. **Notification**
   - Admin receives email: "New form submission"
   - Dashboard shows new intake in queue
   - Can view all form details:
     - Parent/student information
     - Struggles, strengths, interests
     - Program preferences
     - Availability
     - Questions

2. **Review & Triage**
   - Admin reviews intake information
   - Makes notes on:
     - Recommended tier
     - Potential concerns
     - Special accommodations needed
     - Best tutor match
   - Flags for follow-up if needed

3. **Assessment Assignment**
   - Admin or tutor reviews scheduled assessment
   - Tutor assigned based on:
     - Availability match
     - Subject expertise
     - Student age/tier
     - Special needs experience
   - Tutor receives notification with:
     - Student background
     - Parent's concerns
     - Recommended focus areas

### Phase 2: Assessment Conduct

4. **Preparation**
   - Tutor reviews student intake
   - Prepares assessment materials:
     - Grade-appropriate questions
     - Common Core standards checklist
     - Games/activities to use
     - Observation notes template

5. **Assessment Session**
   - Tutor conducts 25-30 minute Zoom session
   - Uses conversational, game-based approach
   - Assesses across subjects:
     - Reading/Language Arts
     - Mathematics
     - Problem-solving
     - Engagement level
   - Takes notes on:
     - Mastery of grade-level standards
     - Learning style observations
     - Strengths demonstrated
     - Areas needing support

6. **Action Plan Creation**
   - Tutor completes assessment form in system:
     - Recommended tier and level
     - Standards mastered vs. needs work
     - Specific strengths identified
     - Areas for improvement
     - Recommended focus skills
     - Suggested learning strategies
   - System generates parent-facing action plan
   - Admin reviews and approves

7. **Results Communication**
   - System sends action plan to parent
   - Tutor may follow up with phone call
   - Answers parent questions
   - Explains recommendations
   - Discusses program options

### Phase 3: Student Assignment & Setup

8. **Profile Finalization**
   - Admin/tutor sets student's:
     - Final tier and level
     - Starting quests
     - Initial XP bonus
     - Character unlocks
   - Assigns tutor to student
   - Sets up communication channels

9. **Tutor Handoff** (if ongoing tutoring)
   - If different tutor will provide ongoing support:
     - Assessment tutor provides notes
     - Ongoing tutor reviews background
     - Introduction session scheduled
   - Tutor prepares personalized approach based on:
     - Student interests
     - Learning style
     - Assessment results

### Phase 4: Ongoing Support & Monitoring

10. **Dashboard Monitoring**
    - Tutor sees on dashboard:
      - All assigned students
      - Recent submissions needing review
      - Students falling behind
      - Milestone achievements
      - Flagged concerns
      - Upcoming sessions

11. **Submission Review (Daily/Every 2 Days)**
    - Reviews student submissions:
      - Text responses
      - Images of work
      - Videos
      - Roblox screenshots
    - Provides feedback:
      - Assigns mastery level
      - Awards XP
      - Writes encouraging comments
      - Identifies misconceptions
      - Suggests improvements
      - Flags for manual review if needed

12. **Live Sessions** (if applicable)
    - Conducts scheduled tutoring sessions:
      - 1-on-1 or small group
      - Via Zoom
      - 25-50 minutes
      - Uses games, Roblox, interactive tools
      - Documents session notes
      - Adjusts difficulty as needed

13. **Progress Tracking**
    - Monitors each student:
      - Quest completion rate
      - Mastery level trends
      - Engagement patterns
      - Areas of struggle
      - Time to complete work
    - Makes adjustments:
      - Quest difficulty
      - Challenge types
      - Support level
      - Communication frequency

### Phase 5: Parent Communication

14. **Regular Updates**
    - Weekly automated reports reviewed
    - Monthly detailed reports compiled
    - Responds to parent messages
    - Initiates communication when:
      - Concerns arise
      - Milestones reached
      - Changes needed
      - Celebrations warranted

15. **Check-in Calls**
    - Scheduled progress check-ins:
      - Monthly or quarterly
      - Discuss growth
      - Adjust goals
      - Celebrate success
      - Address concerns

### Phase 6: Administrative Tasks

16. **Program Management** (Admin)
    - Tracks enrollment:
      - Active students
      - Program types
      - Payment status
      - Tutor assignments
    - Manages scheduling:
      - Tutor availability
      - Student sessions
      - Assessment calendars
    - Handles issues:
      - Payment problems
      - Technical support
      - Parent concerns
      - Tutor support needs

17. **Curriculum Updates**
    - Reviews quest effectiveness
    - Updates challenges based on:
      - Student feedback
      - Tutor observations
      - Learning outcomes
      - Engagement data
    - Creates new content:
      - Seasonal quests
      - Special events
      - New challenges

18. **Quality Assurance**
    - Reviews tutor feedback quality
    - Monitors student satisfaction
    - Analyzes learning outcomes
    - Identifies improvement opportunities
    - Celebrates team successes

## TECHNICAL INTEGRATION FLOW

### Automated Processing Sequence

```
1. Form Submitted (Google Sheets)
   ↓
2. Webhook/Cron Job Triggered
   ↓
3. Google Sheets API Fetch New Rows
   ↓
4. Process Each Submission:
   a. Check if already processed
   b. Save to form_submissions table
   c. Create parent profile (if new)
   d. Create student profile
   e. Create student_intake record
   f. Create assessment record
   g. Mark submission as processed
   ↓
5. Send Notification Emails:
   - Parent: Confirmation + assessment link
   - Admin: New student alert
   - Tutor: Assessment assignment
   ↓
6. Assessment Scheduled
   ↓
7. Assessment Conducted
   ↓
8. Results Entered in System
   ↓
9. Student Profile Finalized
   ↓
10. Parent Receives Action Plan
    ↓
11. Enrollment Decision
    ↓
12. If Enrolled:
    - Payment processed
    - Student dashboard activated
    - Tutor assigned
    - First quest unlocked
    ↓
13. Ongoing Cycle:
    - Student completes work
    - Tutor provides feedback
    - Progress tracked
    - Parents receive updates
```
